# YoungFinance Bergen — Partneroversikt

Live dashboard for sponsorer og pipeline i YoungFinance Bergen.

🔗 **[Åpne dashboardet](https://kevinha98.github.io/yf_partners/)**

## Innhold

Dashboardet gir oversikt over:

- **Aktive sponsorer** — nåværende partnere med beløp, status og avtaleperiode
- **Pipeline** — prospekter under bearbeiding med neste steg og kontaktinfo
- **Historikk** — tidligere sponsorer og avsluttede avtaler
- **Filer** — lenker til pitch decks, maler og sponsoravtaler

## Oppdatering

Rediger `index.html` og push til `main`. GitHub Pages oppdateres automatisk.

## Teknisk

- Statisk HTML/CSS — ingen avhengigheter
- Deployes via GitHub Actions → GitHub Pages
- Font: [Josefin Sans](https://fonts.google.com/specimen/Josefin+Sans) (Google Fonts)
